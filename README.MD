# C (XX) Package Manager
A package manager based around git and inferred builds

## Manifesto
This project is aimed to make C++ more portable and time developers spend with config files. It is inspired by `go get` and Laravel's CLI

## External Dependancies
This project is based entrirely around git and swapping in specific C++ compilers be it `g++`, or `clang`

## Building from Source
```zsh
git clone https://github.com/jrny6996/cxxpm

g++ src/main.cpp  -std=c++20 -o  cxxpm  
```


### Your First Project

`./cxxpm getstarted`
